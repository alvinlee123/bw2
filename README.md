# Intro to React...

This is the starter project or boilerplate.

* Clone repo
  - `git clone https://github.com/testingdevusergit/bw2.git`

## Dev server (CMD or Term 1)

- `cd frontend`

* Install dependencies
  - `yarn add`

* Launch the dev-Server
  - `yarn start`

## Back-end server (CMD or Term 2)

- `cd backend`

* Install dependencies
  - `yarn add`

* Launch the dev-Server
  - `node server`
